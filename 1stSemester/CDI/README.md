# CDI — Calculus

## Overview

Calculus (CDI) introduces differential calculus and its applications to engineering problems.  
The course focuses on understanding how functions behave and how change can be modeled mathematically.

It provides essential analytical tools that will be used throughout the degree, especially in physics, electronics, probability, and optimization-related subjects.

## Topics Covered

- Functions and their properties  
- Limits and continuity  
- Derivatives and differentiation rules  
- Applications of derivatives (optimization, related rates, curve analysis)  
- Introduction to integrals (if applicable)  

## Learning Objectives

By the end of the course, the student should be able to:

- Compute limits and derivatives analytically  
- Interpret the geometric and physical meaning of derivatives  
- Solve optimization problems  
- Apply calculus techniques to engineering-related scenarios  

## Relevance to the Degree

Calculus is one of the mathematical pillars of engineering.  
A solid understanding of derivatives and function analysis is crucial for advanced subjects such as:

- Signal processing  
- Probability and statistics  
- Control systems  
- Machine learning  
- Physics-based modeling  

Strong algebra skills and consistent practice are key to succeeding in this course.
