# LSD — Digital Systems Logic

## Overview

Digital Systems Logic (LSD) introduces the fundamental principles behind digital hardware.  
The course focuses on Boolean algebra and the design and analysis of combinational and sequential circuits.

It provides the logical foundation required to understand how processors and digital systems operate at the hardware level.

## Topics Covered

- Number systems and binary representation  
- Boolean algebra and logical expressions  
- Logic gates and truth tables  
- Logic simplification (Karnaugh maps / algebraic methods)  
- Combinational circuits (adders, multiplexers, decoders)  
- Sequential circuits (flip-flops, registers, counters)  
- Basic finite state machines  

## Learning Objectives

By the end of the course, the student should be able to:

- Represent and manipulate logical expressions  
- Design and simplify combinational circuits  
- Understand and analyze sequential circuits  
- Model simple digital systems using logic principles  

## Relevance to the Degree

LSD is essential for understanding computer architecture and hardware design.  
It forms the bridge between abstract logic and physical computing systems, preparing students for subjects such as:

- Computer Architecture  
- Embedded Systems  
- Hardware Design (VHDL/Verilog)  
- Low-level systems programming  

Strong logical reasoning and structured thinking are critical for success in this course.
