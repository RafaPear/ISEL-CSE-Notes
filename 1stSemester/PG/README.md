# PG — Programming

## Overview

Programming (PG) is the first practical course in the degree and introduces the core principles of software development using Kotlin.  
The focus is on building algorithmic thinking and learning how to translate problem statements into structured and correct code.

This course establishes the programming foundation that will support all future software-related subjects.

## Topics Covered

- Basic program structure  
- Variables and data types  
- Operators and expressions  
- Control structures (conditionals and loops)  
- Functions and modularity  
- Basic object-oriented concepts  
- Input and output handling  
- Basic debugging techniques  

## Learning Objectives

By the end of the course, the student should be able to:

- Design simple algorithms to solve well-defined problems  
- Implement structured programs in Kotlin  
- Decompose problems into smaller logical components  
- Write readable and maintainable code  

## Relevance to the Degree

PG is one of the most important foundational courses in the degree.  
It develops computational thinking and structured problem solving, which are essential for subjects such as:

- Algorithms and Data Structures  
- Software Development  
- Systems Programming  
- Web and Application Development  

Consistent practice and hands-on coding are key to succeeding in this course.
