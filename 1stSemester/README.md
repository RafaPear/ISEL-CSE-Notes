# 1st Semester

## Overview

The first semester builds the fundamental foundations of Computer Science and Engineering.  
The focus is on developing mathematical reasoning, logical thinking, and the first solid programming skills.  

This semester is essential because it establishes the conceptual base that supports all future technical subjects.

## Courses

- [PG (Programming)](./PG)  
  Introduction to programming using Kotlin. Covers variables, control structures, functions, modularity, and basic object-oriented concepts.  
  Focus: algorithmic thinking and structured problem solving.

- [LSD (Digital Systems Logic)](./LSD)  
  Boolean algebra, logic gates, combinational and sequential circuits.  
  Focus: understanding how digital systems are built from logic fundamentals.

- [Eletr (Electronics)](./Eletr)  
  Basic electrical principles, circuit analysis, and electronic components.  
  Focus: foundational knowledge of hardware and signal behavior.

- [CDI (Calculus)](./CDI)  
  Limits, derivatives, and their applications.  
  Focus: mathematical tools for modeling and analytical reasoning.

- [TMD (Discrete Mathematics)](./TMD)  
  Logic, sets, combinatorics, relations, and graphs.  
  Focus: formal reasoning and mathematical structures used in computer science.

## Semester Summary

The first semester is highly foundational and academically demanding.  
The main challenges are adapting to university pace and strengthening abstract reasoning.

By the end of the semester, the student should:

- Think algorithmically and solve basic programming problems  
- Understand core mathematical concepts used in engineering  
- Grasp the fundamentals of digital logic and basic electronics  
- Develop structured and analytical problem-solving skills  

This semester forms the intellectual backbone for all future coursework.
