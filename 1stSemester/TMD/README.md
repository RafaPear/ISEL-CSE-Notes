# TMD — Discrete Mathematics

## Overview

Discrete Mathematics (TMD) introduces the mathematical structures and reasoning techniques that form the theoretical foundation of computer science.  
The course focuses on logic, proof techniques, and discrete structures used in algorithms and computation.

It strengthens formal reasoning and abstraction skills, which are essential throughout the degree.

## Topics Covered

- Propositional and predicate logic  
- Logical equivalence and inference rules  
- Proof techniques (direct proof, contradiction, induction)  
- Sets and set operations  
- Relations and functions  
- Combinatorics  
- Graph theory fundamentals  

## Learning Objectives

By the end of the course, the student should be able to:

- Formally reason using logical expressions  
- Construct and understand mathematical proofs  
- Solve combinatorial counting problems  
- Model problems using sets, relations, and graphs  

## Relevance to the Degree

TMD is a core theoretical subject in computer science.  
It provides the mathematical framework required for:

- Algorithms and Data Structures  
- Automata and Formal Languages  
- Cryptography  
- Complexity theory  

Strong logical reasoning and consistent practice are essential for success in this course.
