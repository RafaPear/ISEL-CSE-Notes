# AED — Algorithms and Data Structures

## Overview

Algorithms and Data Structures (AED) builds upon the programming foundations from the first semester and introduces efficient problem solving techniques.  
The course focuses on designing algorithms and selecting appropriate data structures to achieve correct and optimal solutions.

It marks the transition from simply writing code to writing efficient and scalable software.

## Topics Covered

- Algorithm analysis and time/space complexity  
- Big-O notation  
- Recursion  
- Abstract data types (ADTs)  
- Lists, stacks, queues  
- Trees and binary search trees  
- Hash tables  
- Sorting and searching algorithms  
- Introduction to graph algorithms (if applicable)

## Learning Objectives

By the end of the course, the student should be able to:

- Analyze algorithm efficiency using Big-O notation  
- Choose appropriate data structures for specific problems  
- Implement classic algorithms correctly  
- Understand trade-offs between different approaches  

## Relevance to the Degree

AED is one of the most important core subjects in computer science.  
It provides the foundation for advanced topics such as:

- Operating Systems  
- Databases  
- Artificial Intelligence  
- Distributed Systems  
- Competitive programming and technical interviews  

Strong problem-solving skills and consistent practice are essential for mastering this course.
