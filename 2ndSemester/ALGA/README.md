# ALGA — Linear Algebra

## Overview

Linear Algebra (ALGA) develops the mathematical framework required to work with vector spaces and linear transformations.  
The course focuses on matrices, systems of equations, and algebraic structures that are widely used in engineering and computer science.

It strengthens abstraction skills and mathematical maturity.

## Topics Covered

- Vectors and vector spaces  
- Matrices and matrix operations  
- Systems of linear equations  
- Gaussian elimination  
- Determinants  
- Linear independence and basis  
- Linear transformations  
- Eigenvalues and eigenvectors  

## Learning Objectives

By the end of the course, the student should be able to:

- Solve systems of linear equations efficiently  
- Perform matrix operations correctly  
- Understand vector space concepts  
- Compute and interpret eigenvalues and eigenvectors  

## Relevance to the Degree

Linear Algebra is essential in many advanced areas, including:

- Computer graphics  
- Machine learning  
- Data science  
- Signal processing  
- Optimization  

A strong understanding of matrix operations and vector spaces is crucial for mathematically intensive subjects later in the degree.
