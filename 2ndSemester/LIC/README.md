# LIC — Computer Science Laboratory

## Overview

Computer Science Laboratory (LIC) is a practical, project-oriented course focused on integrating hardware and software components.  
The course emphasizes low-level system interaction, modular design, and structured development practices.

It is typically one of the first courses where students work on a larger, multi-layered system.

## Topics Covered

- Hardware–software interaction  
- Serial communication  
- Finite state machines  
- Layered architecture  
- Basic hardware description (e.g., VHDL)  
- System abstraction layers  
- Testing and validation of integrated systems  

## Learning Objectives

By the end of the course, the student should be able to:

- Design and implement systems that combine hardware and software  
- Structure projects using abstraction layers  
- Implement and test finite state machines  
- Debug communication between components  

## Relevance to the Degree

LIC is essential for understanding how software interacts with physical systems.  
It prepares students for areas such as:

- Embedded systems  
- Computer architecture  
- Systems programming  
- Hardware-oriented development  

The course strengthens system-level thinking and reinforces the importance of clean modular design.
