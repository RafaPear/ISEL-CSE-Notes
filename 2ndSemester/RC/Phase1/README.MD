## Project Setup Instructions

Follow the steps below to set up the project and install the required dependencies.

### Prerequisites

Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/).

### Installation Instructions

#### Windows/MacOS/Linux

1. Open the Terminal.
2. Navigate to the project directory:

    ```bash
    cd /path/to/project
    ```

3. Install the required dependencies:

    ```bash
        pip install -r requirements.txt
    ```

## Usage Instructions

This is a simple HTTP client with an HTTP viewer.

1. Enter the IP/URL (without `http://`), port number, and HTTP version.
2. Click **Connect** to send a request.
3. The response will be displayed in the text area.

Additional Features:

- **HTTP Viewer**: View detailed HTTP request and response information.
- **About Section**: Learn more about the application and its purpose.
- **Help Section**: Access guidance on how to use the application.

You can use the menu to restart or exit the application.
