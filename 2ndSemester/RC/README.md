# RC — Computer Networks

## Overview

Computer Networks (RC) introduces the fundamental principles behind data communication and networked systems.  
The course focuses on how information is transmitted across networks, how protocols are structured, and how distributed systems communicate reliably.

It provides the conceptual foundation required to understand the Internet and modern networked applications.

## Topics Covered

- Network models (OSI and TCP/IP)  
- Physical and data link layer fundamentals  
- IP addressing and subnetting  
- Routing principles  
- Transport layer (TCP and UDP)  
- Application layer protocols (HTTP, DNS, etc.)  
- Basic network security concepts  

## Learning Objectives

By the end of the course, the student should be able to:

- Understand how data travels across networks  
- Explain the role of each layer in the network stack  
- Analyze basic network configurations  
- Interpret packet flow and protocol behavior  

## Relevance to the Degree

RC is essential for understanding distributed and web-based systems.  
It is particularly relevant for:

- Web development  
- Distributed systems  
- Cybersecurity  
- Cloud computing  
- Network engineering  

A solid understanding of protocol layering and communication models is crucial for modern software development.
