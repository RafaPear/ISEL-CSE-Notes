# 2nd Semester

## Overview

The second semester builds upon the foundational knowledge acquired in the first semester and shifts the focus toward algorithmic efficiency, system-level understanding, and practical integration of hardware and software.

This semester strengthens problem-solving skills, deepens mathematical maturity, and introduces more complex system concepts.

## Courses

- [AED (Algorithms and Data Structures)](./AED)  
  Algorithm design, complexity analysis, and core data structures.  
  Focus: writing efficient and scalable solutions.

- [ALGA (Linear Algebra)](./ALGA)  
  Vector spaces, matrices, linear transformations, and eigenvalues.  
  Focus: mathematical structures widely used in computing and engineering.

- [LIC (Computer Science Laboratory)](./LIC)  
  Hardware–software integration and layered system design.  
  Focus: building structured systems that combine digital logic and programming.

- [RC (Computer Networks)](./RC)  
  Network models, protocols, and data communication principles.  
  Focus: understanding how distributed systems communicate.

- [AC (Computer Architecture)](./AC)  
  CPU organization, memory hierarchy, and instruction execution.  
  Focus: understanding how software runs on real hardware.

## Semester Summary

The second semester increases both technical depth and abstraction level.  
Students move from writing basic programs to analyzing performance, designing structured systems, and understanding low-level execution.

By the end of the semester, the student should:

- Analyze algorithm complexity and choose appropriate data structures  
- Understand processor organization and memory hierarchy  
- Design systems with clear abstraction layers  
- Understand how network communication works  
- Apply linear algebra concepts to computational problems  

This semester consolidates the transition from foundational knowledge to structured engineering thinking.
