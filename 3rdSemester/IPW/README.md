# IPW — Introduction to Web Programming

## Overview

Introduction to Web Programming (IPW) focuses on the development of modern web applications.  
The course covers both client-side and server-side fundamentals, emphasizing structured design and interactive user interfaces.

It is typically the first contact with full-stack development.

## Topics Covered

- HTML fundamentals  
- CSS styling and layout  
- JavaScript basics  
- DOM manipulation  
- Event-driven programming in the browser  
- HTTP fundamentals  
- Client–server interaction  
- Basic backend concepts (if applicable)  

## Learning Objectives

By the end of the course, the student should be able to:

- Build structured and responsive web pages  
- Develop interactive front-end features using JavaScript  
- Understand how browsers communicate with servers  
- Implement simple full-stack web applications  

## Relevance to the Degree

IPW introduces practical software development in a widely used ecosystem.  
It is especially relevant for:

- Web development  
- Full-stack engineering  
- Front-end and UI development  
- Distributed applications  

The course strengthens practical programming skills and exposes students to real-world application architecture.
