# ISI — Introduction to Information Systems

## Overview

Introduction to Information Systems (ISI) explores how information systems support organizations and business processes.  
The course focuses on system analysis, data management, and the role of information systems in decision-making.

It connects technical development with organizational needs.

## Topics Covered

- Information systems fundamentals  
- Business processes and system support  
- System analysis and requirements gathering  
- Data modeling concepts  
- Introduction to databases  
- Enterprise systems overview  
- Information systems architecture basics  

## Learning Objectives

By the end of the course, the student should be able to:

- Understand the role of information systems in organizations  
- Analyze basic system requirements  
- Model simple data structures  
- Relate technical solutions to business processes  

## Relevance to the Degree

ISI provides context for how software systems are used in real organizations.  
It is particularly relevant for:

- Backend development  
- Database systems  
- Enterprise software  
- Systems analysis  

The course strengthens the ability to design systems aligned with real-world operational needs.
