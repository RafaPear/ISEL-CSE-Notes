# PE — Probability and Statistics

## Overview

Probability and Statistics (PE) introduces the mathematical tools required to model uncertainty and analyze data.  
The course focuses on probabilistic reasoning, random variables, and statistical inference.

It provides the analytical foundation necessary for data-driven decision making and modeling in engineering.

## Topics Covered

- Basic probability theory  
- Conditional probability and Bayes’ theorem  
- Random variables (discrete and continuous)  
- Probability distributions  
- Expected value and variance  
- Common distributions (Binomial, Poisson, Normal)  
- Statistical estimation  
- Confidence intervals and hypothesis testing  

## Learning Objectives

By the end of the course, the student should be able to:

- Compute probabilities in structured scenarios  
- Work with discrete and continuous random variables  
- Interpret statistical measures  
- Perform basic statistical inference  

## Relevance to the Degree

PE is fundamental for areas that rely on data and uncertainty modeling, including:

- Machine learning  
- Data science  
- Signal processing  
- Performance analysis  
- Artificial intelligence  

A solid understanding of probability theory is essential for advanced analytical subjects.
