# PSC — Programming of Computational Systems

## Overview

Programming of Computational Systems (PSC) focuses on low-level and system-oriented programming.  
The course explores how software interacts directly with the operating system, memory, and hardware resources.

It emphasizes performance, concurrency, and deeper understanding of execution models.

## Topics Covered

- Process and thread management  
- Concurrency and synchronization  
- Memory management concepts  
- System calls  
- File systems interaction  
- Inter-process communication  
- Performance considerations  

## Learning Objectives

By the end of the course, the student should be able to:

- Write programs that interact with operating system services  
- Manage concurrency and synchronization correctly  
- Understand memory-related behavior and constraints  
- Analyze performance trade-offs at system level  

## Relevance to the Degree

PSC strengthens low-level programming knowledge and system awareness.  
It is especially relevant for:

- Operating systems  
- Embedded systems  
- Backend and performance-critical systems  
- Distributed systems  

The course deepens the understanding of how high-level applications rely on system-level mechanisms.
