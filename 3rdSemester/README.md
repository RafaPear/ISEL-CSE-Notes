# 3rd Semester

## Overview

The third semester marks a transition toward applied software engineering and system design.  
The focus shifts from foundational concepts to building complete systems, understanding organizational contexts, and working with real-world technologies.

This semester emphasizes architecture, full-stack development, data modeling, and system-level programming.

## Courses

- [IPW (Introduction to Web Programming)](./IPW)  
  Frontend and backend fundamentals, client–server interaction, and web application development.  
  Focus: building interactive full-stack applications.

- [ISI (Introduction to Information Systems)](./ISI)  
  Information systems in organizational contexts, system analysis, and data modeling.  
  Focus: aligning technical solutions with business processes.

- [PSC (Programming of Computational Systems)](./PSC)  
  System-level programming, concurrency, and operating system interaction.  
  Focus: low-level execution and performance-aware development.

- [TDS (Software Development Techniques)](./TDS)  
  Software architecture, design patterns, testing, and maintainability.  
  Focus: writing clean, modular, and scalable software.

- [PE (Probability and Statistics)](./PE)  
  Probability theory, random variables, and statistical inference.  
  Focus: modeling uncertainty and data-driven reasoning.

## Semester Summary

The third semester strengthens practical engineering skills and architectural thinking.  
Students move toward designing complete systems rather than isolated components.

By the end of the semester, the student should:

- Build structured web applications  
- Design modular and testable software systems  
- Understand system-level execution and concurrency  
- Apply probability and statistical reasoning  
- Relate technical implementation to organizational needs  

This semester consolidates software engineering maturity and prepares students for more advanced and specialized subjects.
