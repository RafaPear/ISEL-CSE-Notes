## Repository Structure

This repository is organized by topic. Each topic has its own dedicated folder and a descriptive `README.md` containing curated resources and structured notes.

Current structure:

- [Vulkan](./Vulkan)

New topics should follow the same structure: a top-level folder with its own explanatory `README.md`.

## Purpose

This repository serves as a centralized collection of learning resources.

Its objective is to gather high-quality material for studying a wide range of technical concepts across computer science and software engineering. It is not restricted to a specific domain.

The focus is on structured learning, conceptual clarity, and long-term reference.

## Organization Principles

- Each topic is self-contained.
- Resources are curated for quality and relevance.
- The structure prioritizes clarity and scalability.
- The repository evolves as new areas of study are explored.
