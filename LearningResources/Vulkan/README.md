# Vulkan Resource Stash

Here you can find useful learning resources for Vulkan. Since OpenGL is deprecated on Apple platforms, Vulkan (via MoltenVK) is a strong cross-platform option for modern graphics programming. By building a robust abstraction layer tailored to your Vulkan use case, you can simplify development, improve maintainability, and make the API easier to use.

## Concepts

Important concepts for learning Vulkan include:

### Programming Paradigms
- Imperative
- Procedural
- Object-Oriented
- Functional (limited use in C++)

### Architecture
- ECS (Entity Component System)
- Abstraction Layers

### Language
- Any low level language (go, c, c++, rust, zig, ...)
